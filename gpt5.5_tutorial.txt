i downloaded the jsx file - but how do i run it 

You downloaded a React component, not a standalone HTML file. It needs a React runtime and build system.

The cleanest path is a modern React/Vite setup.

1. Install Node.js
Install the current LTS version from:

Node.js

Then verify:

node -v
npm -v
2. Create a Vite React project
Open terminal in your desired directory:

npm create vite@latest bird-wing-physics -- --template react
Then:

cd bird-wing-physics
npm install
3. Install dependencies used by the artifact
The component uses:

framer-motion

recharts

lucide-react

Tailwind-style utility classes

shadcn/ui-style imports

Install the runtime libraries:

npm install framer-motion recharts lucide-react
4. Replace the default app
Inside:

src/
replace:

App.jsx
with your downloaded JSX file.

5. Remove shadcn imports (important)
Your artifact currently imports:

@/components/ui/card
@/components/ui/button
@/components/ui/slider
@/components/ui/tabs
@/components/ui/badge
Those exist only in projects already configured with shadcn/ui.

Fastest solution:

replace those imports with lightweight inline components.

Add this near the top of the file after imports:

function Card({ children, className="" }) {
  return <div className={className}>{children}</div>;
}

function CardContent({ children, className="" }) {
  return <div className={className}>{children}</div>;
}

function Button({ children, className="", variant }) {
  return (
    <button className={className}>
      {children}
    </button>
  );
}

function Badge({ children, className="" }) {
  return (
    <span className={className}>
      {children}
    </span>
  );
}
Then DELETE these import lines:

import { Card, CardContent } from "@/components/ui/card";
import { Button } from "@/components/ui/button";
import { Slider } from "@/components/ui/slider";
import { Badge } from "@/components/ui/badge";
import { Tabs, TabsContent, TabsList, TabsTrigger } from "@/components/ui/tabs";
6. Simplify the slider component
The current code assumes shadcn slider.

Replace:

<Slider ... />
with native HTML sliders:

Find:

<Slider value={[value]} ...
Replace with:

<input
  type="range"
  value={value}
  min={min}
  max={max}
  step={step}
  onChange={(e) => onChange(Number(e.target.value))}
  className="w-full"
/>
Now no UI library is needed.

7. Temporarily remove Tabs
The tabs also depend on shadcn.

Fastest path:

Delete:

import { Tabs, TabsContent, TabsList, TabsTrigger } ...
Then temporarily replace the entire <Tabs> block with a simple div:
